「智慧栾川」 · 洛阳市生活服务平台

  • 项目简介：外包项目，投标获得，洛阳市政府为扩展旅游业推行的线上服务平台，在该平台上，用户可查找超市、物流门店、学校、医院、进行生活缴费、浏览招聘和二手交易信息等本地服务，可在微圈中进行图片、视频分享等线上交流，也可预订旅游景点的门票。
  • 负责内容：
    • 负责「微圈」的模块设计、编码和调试。
    • 负责 iOS 和 Web 的 JavaScript 通信。
    • 支付模块设计、编码和调试。
  • 技术难点：自定义视频录制
    • 采用 AVFoundation 框架的 AVCaptureSession 实现小视频的录制，AVAssetExportSession 对视频进行压缩，并采用 AVAsset 截取视频第一帧图片。
    • 视频的输出对比了 AVCaptureMovieFileOutput 和 AVAssetWriter，AVCaptureMovieFileOutput 只需要一个输出，视频和音频会写入到指定路径，不需要其他复杂操作；AVAssetWriter 的可配置参数比较多，但需要音频和视频两个单独的输出，然后进行相应的处理。
    • 综合项目需求和成本的考虑，选择了 AVCaptureMovieFileOutput 来处理视频的输出。

「党员积分」 · 国家电网党员绩效考核平台

  • 项目简介：外包项目，国家电网洛阳分公司用于进行党员考核管理的APP。在该 APP 中，按照党组织的职务等级对用户进行权限划分，主要分为党委书记、党建管理员、支部书记、小组组长、普通党员5个等级，组长可对本组党员进行绩效打分，及日常会议活动的灵活评分，由所在支部的书记进行审核；普通党员和组长可在线申请填报高级职称和各种省市奖项，由书记审核后更新分数；组长以上领导层可发布会议计划，推送通知到所有参会人员，参会人员在规定时间内进行签到。
  • 负责内容：负责「党员积分」APP 的架构设计、编码、调试、上线等。
  • 技术难点：UITabBarItem、UIBarButtonItem、UIView 等控件的自定义角标实现。
    • 利用 Reveal 分析 UI 布局，找到 UITabBarItem 和 UIBarButtonItem 中的文字视图，作为角标控件的父视图；
    • 通过协议创建属性、声明方法；
    • 创建类别，实现协议中角标的显示、更新等方法，利用 Runtime 机制动态绑定成员变量。
  • 技术亮点：利用 Runtime 机制实现字体的全局设置，对不同机型的屏幕进行适配。
    • 为 UIFont 创建类别，将 systemFontOfSize 方法替换为自定义的方法，在其中将字体大小 + 2（判断屏幕尺寸确定是否改变字体）；
    • 利用 Runtime 机制将 UILabel 的 initWithCoder 方法换成自定义的方法，在其中调用 systemFontOfSize 方法重设字体大小；

「我们约会吧」 · 交友类 APP

  • 项目简介：交友类 APP，抽屉式导航，应用首页通过雷达扫描展示附近用户，用户通过左滑或右滑操作对对方进行like / unlike标记；左侧抽屉是菜单栏，右侧抽屉是好友列表；聊天消息类型包括文字、表情、音频、图片、位置等。
  • 负责内容：负责整体项目架构的设计、编码、调试、上线等。
  • 技术难点：聊天界面中双方消息的 cell 展示（图文混排）。
    • 本人和对方的消息分别出现在左、右两侧，由于数据模型是一致的，通过继承方式来实现左、右两侧的布局处理；
    • 通过自动布局将文本、音频、图片、位置四类消息视图从上到下排列，如某消息是文本类，则将文本类的高度约束按文字内容计算后设置，其余消息视图高度约束设为0.
  • 技术亮点：聊天界面，进行音频录制时，通过 UIButton 的各种 event 事件实现点击开始录制、松开结束录制、上滑暂停录制、下滑继续录制、上滑后松开取消录制等操作。
